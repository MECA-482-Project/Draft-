Make sure you have following files in your directory
1. sim.py
2. simConst.py
3. the appropriate remote API library: "remoteApi.dll" (Windows), "remoteApi.dylib" (Mac) or "remoteApi.so" (Linux)
4. simpleTest.py (or any other example file)

The uploaded files are needed in order to interface python code into coppelia sim models
